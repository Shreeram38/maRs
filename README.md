# NASA Mars API
To retrieve and visualize weather data from the last 7 sols (martian days) on mars.

# Installation:
clone repository

`git clone https://github.com/claudia-nikel/534_project/`

in command line:

`R CMD build maRs`

`R CMD INSTALL maRs_0.1.tar.gz`

in R console:

`library(maRs)`
